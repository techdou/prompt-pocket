What's inside

克制，但每一个像素都经过打磨。

一个专注的提示词口袋：只在需要时出现，复制完即消失，把常用 Prompt 变成肌肉记忆。

01 / 唤出
⚡

全局秒唤，失焦即收

Ctrl+Alt+P 从任意应用唤出 spotlight 窗口，复制完自动隐藏，失焦也会收起。不抢焦点、不占任务栏，像系统自带的搜索一样轻盈。

02 / 搜索
🔍

模糊搜索 · 键盘全程可达

标题 / 分类 / 正文实时检索，Ctrl+F 聚焦、↑↓ 选择、Enter 复制——从唤出到粘贴，一次都不必碰鼠标。

03 / 存储
📝

Markdown 文件 + 填空编辑

一条 Prompt 一个 .md，文件夹即分类。标题/分类用表单填，无需手写 YAML。

04 / 同步
☁️

坚果云 WebDAV 多端同步

内置设置界面，填账号 + 应用密码即可。启动自动拉取、保存自动推送。

05 / 排序
🔀

原生拖拽排序

按住列表项手柄拖到目标位置，落点指示线实时跟随。

06 / 轻量
🪶

1.6MB 安装包，~42MB 内存

Tauri v2 + Rust 后端，无 Electron、无内嵌浏览器。没有数据库，没有云端账号——数据完全归你所有，用你自己的坚果云同步到所有设备。CPU 占用近乎为零，常驻后台毫无负担。

Get it

Windows 用户，下载即用。

提供 NSIS 与 MSI 两种安装包，双击安装，无需额外运行环境。

⬇ Windows 安装程序 NSIS · .exe · 约 2.7 MB ⬇ MSI 安装包 .msi · 约 3.8 MB

其他平台或想自行编译？看下面的源码步骤，或前往 全部 Releases。

从源码构建（需 Node.js 18+ 与 Rust 1.77+）
bash — prompt-pocket 
# 克隆并安装依赖
git clone https://github.com/techdou/prompt-pocket.git
cd prompt-pocket
npm install

# 开发模式（热重载）或打包发布
npm run tauri:dev
npm run tauri:build   # 产物在 src-tauri/target/release/bundle/

别再翻聊天记录找提示词了。
